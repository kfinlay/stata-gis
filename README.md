stata-gis
=========

List of GIS tools for Stata

Geodata conversion
  * `shp2dta`
  * `mif2dta`
  * `dta2ras`
  * `ras2dta`

Vector data functions
  * `mergepoly`
  
Geocoding
  * `geocode3`
  * `traveltime`
  * `geocodeopen`

Distance functions
  * `distmatch`
  * `geonear`
  * `vincenty`

Mapping
  * `spmap`
  * `geochart`
  * `kmlmap`
  * `spgrid`
  * `tmap`
  * `worldstat`
  
Raster data analysis
  * `dta2ras`
  * `ras2dta`
  * `spatsam`
